# Quick Help Syntax

## Quick Help Syntax(1)
> 한 줄의 주석은 세 개의 슬래시 (/) 문자를 사용하세요.

```
/// line of markup content
```

## Quick Help Syntax(2)
> 여러줄의 주석에서 여는 주석에는 슬래시 (/) 다음에 별표 두 개(**)를 사용하고 닫는 주석에는 별표 다음 슬래시를 사용하세요.

```
/**
    line of text with optional markup
    …
    line of text with optional markup
*/
```
